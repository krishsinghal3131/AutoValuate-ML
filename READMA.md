# AutoValuate-ML 🚗💰

A high-performance Machine Learning model built from scratch to predict used car selling prices. This project focuses on end-to-end data pipelines, categorical encoding, and regression analysis.

## 📌 Project Overview
Predicting the market value of a used car can be complex due to multiple interacting factors. This repository implements two core regression algorithms to analyze automobile specifications and predict optimal selling prices with high accuracy.

## 🚀 Key Features & Workflow
- **Data Engineering:** Handled categorical variables (Fuel Type, Seller Type, Transmission) using numeric mapping.
- **Predictive Modeling:** Implemented and compared both **Linear Regression** and **Lasso Regression** (L1 Regularization).
- **Evaluation:** Validated performance metrics, achieving a strong training accuracy.
- **Data Visualization:** Built scatter plots comparing Actual vs. Predicted prices to evaluate model fit visually.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib

## 📊 Performance Metrics
- **Linear Regression $R^2$ Score:** `0.8799` (~88% variance explained)
- **Lasso Regression $R^2$ Score:** `0.8709`
